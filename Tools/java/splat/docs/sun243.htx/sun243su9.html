3.6 Spectral lines query

To start the spectral lines browser window, click on the symbol from a plot. In this window, spectral lines can be searched using SLAP and VAMDC services and displayed into a plot. First of all, one or more wavelength ranges have to be added to the ranges list by clicking on the Add Button and selecting a region in the plot. The selected region will be marked by a gray rectangle. After the query, or when it’s not needed anymore, it can be removed from the plot using the Delete button. To search use this window with a different plot, use the plot selection box.

The query options that can be added are element symbol and stage/ion charge. Most SLAP services do not support (yet) the later, but it’s supported by VAMDC services.

The selection of the services to be queried is similar to SSAP and Obscore browsers intrfaces. In this case, there are two services tabs, one for SLAP services, and other for VAMDC services.

The Search Button sends the query to the selected services. The results will appear on the results area on the right side of the window. To display selected lines on the plot use the use the buttons on the lower right side, or right-click on the line you want to display. The Remove lines button removes all lines from the plot.

The lines browser window is linked to the plot which started it. Wavelength range selection can be done only in this plot, as well as spectral lines can be only displayed in this plot.

SLAP (Simple Line Access Protocol) is being developed by the IVOA (International Virtual Observatory Alliance3), and VAMDC services are provided by the (VAMDC Consortium4).
